beatbits beatmaker

Beatbits is a website that allows users to make small bits of music using a drum sequencer. It uses Django for the backend, PostgreSQL for the database, vanilla JS for the frontend, and Tone.js to control sound output. It is deployed on Heroku.

polyretro synthesizer

This website is a polyphonic synthesizer for creating music. It is built with React and Tone.js. It is deployed on Github Pages.

utrails trail finder

Utrails is a website you can use to find and/or review over 500 hiking and biking trails in Utah. It uses a Flask backend and a PostgreSQL database. It is deployed on Heroku.

league of legends build generator

This website is a League of Legends build generator. It uses Riot Games API to pull in data on champions and items. It uses vanilla JS to generate a random build based on a random champion. It is deployed on Github Pages.

social network site demo

This is a demo of a social network site. It uses Django for the backend and a PostgreSQL database. It is deployed on Heroku.
